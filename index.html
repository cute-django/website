Welcome to cute-django!

The aim of cute-django is to create a full framework able to encapsulate, build and deploy a normal Django application in a stand-alone desktop application.

The project is based on the following powerful technologies:

  • Django Framework
  • Qt Framework
  • Qt Installer Framework
  • PyInstaller

The project is completed free and open source: join us!
